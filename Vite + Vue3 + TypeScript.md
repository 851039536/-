# Vite + Vue3 + TypeScript

### 使用 Vite 快速初始化项目雏形

- 使用 NPM：

  ```bash
  npm init @vitejs/app
  ```

- 使用 Yarn：

  ```bash
  yarn create @vitejs/app
  ```

### 选择模板

本项目需要使用 Vue3 + TypeScript，所以我们选择 `vue-ts`，会自动安装 Vue3 和 TypeScript。

1. 安装依赖

   ```bash
   npm install
   ```

2. 启动项目

   ```bash
   npm run dev
   ```
